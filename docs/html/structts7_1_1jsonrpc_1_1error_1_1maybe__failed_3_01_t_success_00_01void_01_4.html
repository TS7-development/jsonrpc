Detailed Description

template<typename TSuccess>
struct ts7::jsonrpc::error::maybe_failed< TSuccess, void >

Maybe failed.

Represents a return type, that can fail or succeed. If succeeded the original return value of the function is accessable. There is no return value stored in case of failure.

Template Parameters
TSuccessData type, if succeeded.

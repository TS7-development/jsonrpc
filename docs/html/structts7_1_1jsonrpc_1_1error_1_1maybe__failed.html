Detailed Description

template<typename TSuccess, typename TFailed>
struct ts7::jsonrpc::error::maybe_failed< TSuccess, TFailed >

Maybe failed.

Class that represents a return type that maybe has failed. This stores the success value, if the function succeeded. Otherwise it stores the failed return type, if the function has failed.

Note
TSuccess and TFailed need to support the default constructor.
Template Parameters
TSuccessThe return type, if the function succeeded.
TFailedThe return type, if the function failed.

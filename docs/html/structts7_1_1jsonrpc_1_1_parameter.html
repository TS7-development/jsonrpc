Retrieves the required parameter from the json object, if the json object contains it. Otherwise it is returning the default value, if the parameter has a default value.

Returns the retrieved value, if the parameter was present in the json object. If not the default value will be returned, if available. Otherwise it will for now just return the default constructor.

template<typename TReturn, typename T1, typename T2>
struct ts7::jsonrpc_playground::json_call::Test_v2< TReturn, T1, T2 >

Test version 2.

This test is an improved version of Test_v1. In this test we stick with the parameter amount fixed to 2 and also still use fixed names for the parameters. But the return type as well as the two parameter types can now be templated. Everything aside of it, is still working as in Test_v1.

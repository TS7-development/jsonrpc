boost::any storage

Stores instances of boost:any and is capable of calling member functions to those.

Add any type.

Adds an instance of any type, defined by U.

Note
This function is not only the instance under it's name in a map. Also a lambda is created that casts the any type to it's original type and calls the member function print() on it, stored by it's type id hash.
Template Parameters
UThe type of the instance that shall be stored.
Parameters
nameName of the element to store, like a JSON RPC method name.
uThe forward reference of an instance, that shall be stored.

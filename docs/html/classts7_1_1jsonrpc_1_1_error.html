Detailed Description

template<typename TId>
class ts7::jsonrpc::Error< TId >

Error message.

Represents an error message generator.

Note
According to the specification an error does not contain the method name. If the method name is wished, it can be enabled with the define TS7_JSONRPC_SUPPORT_RESPONSE_METHOD_NAME.

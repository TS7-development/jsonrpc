Detailed Description

template<typename TFailed>
struct ts7::jsonrpc::error::maybe_failed< void, TFailed >

Maybe failed.

Represents a return type that can have maybe failed. This template specialization only supports failure information, but no success information.

Template Parameters
TFailedData type of the failure information.

Add any type. 

Adds an instance of any type, defined by {\ttfamily U}.

\begin{DoxyNote}{Note}
This function is not only the instance under it\textquotesingle{}s name in a map. Also a lambda is created that casts the any type to it\textquotesingle{}s original type and calls the member function \mbox{\hyperlink{structts7_1_1jsonrpc_1_1playground_1_1any__collection_1_1_storage_aa16812579563206a91682898018ca5e7}{print()}} on it, stored by it\textquotesingle{}s type id hash.

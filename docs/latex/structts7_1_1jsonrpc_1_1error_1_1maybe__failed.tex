struct ts7\+::jsonrpc\+::error\+::maybe\+\_\+failed$<$ TSuccess, TFailed $>$}

Maybe failed. 

Class that represents a return type that maybe has failed. This stores the success value, if the function succeeded. Otherwise it stores the failed return type, if the function has failed.

\begin{DoxyNote}{Note}
TSuccess and TFailed need to support the default constructor.
\end{DoxyNote}

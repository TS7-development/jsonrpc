From Json Converter. 

This template is meant to convert from a json value to any other type. Every supported type needs it\textquotesingle{}s own specialization. All unsupported types will use the unspecialized template, which will fail a static\+\_\+assert in its constructor. This way you will see at compile time, that the software does not know how to convert from json value to your specific type.

\begin{DoxyNote}{Note}
The error message will anywhere contain \char`\"{}\mbox{[}\+T = Dummy\mbox{]}\char`\"{} (assuming the type was \mbox{\hyperlink{structts7_1_1jsonrpc__playground_1_1json__call_1_1_dummy}{Dummy}}). The exact wording and format is compiler dependend.
\end{DoxyNote}

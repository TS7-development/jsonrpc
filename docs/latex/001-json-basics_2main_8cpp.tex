JSON basics. 

Goal of this example is to figure, how to use the boost\+::json library. Most crucial information is the capability of providing object references, to avoid constantly updating parents when an object has changed. This was required in Qt 5.\+11 and was super annoying and has lead to a lot of avoidable errors.

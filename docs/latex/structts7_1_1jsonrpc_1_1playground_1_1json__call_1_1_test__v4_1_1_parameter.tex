Call operator. 

Retrieves the required parameter from the json object, if the json object contains it. Otherwise it is returning the default value, if the parameter has a default value.

\begin{DoxyAttention}{Attention}
If the object does not contain the parameter and has no default value, there should be an error handling any how. This was omitted to keep the example simple. There is still no type checking for the parameter.
\end{DoxyAttention}

\begin{DoxyParams}{Parameters}
{\em o} & Json object that contains the provided parameter.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Returns the retrieved value, if the parameter was present in the json object. If not the default value will be returned, if available. Otherwise it will for now just return the default constructor.
\end{DoxyReturn}
